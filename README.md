# Discrete Solution Operator Learning (DiSOL)

This repository contains the code and trained models for the Discrete Solution Operator Learning (DiSOL) framework.

## Overview

DiSOL is a neural operator framework designed for geometry-dependent partial differential equations (PDEs).
