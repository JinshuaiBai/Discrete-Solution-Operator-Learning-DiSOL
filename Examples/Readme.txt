Commands for training and testing are provided in each folder.

Training and Testing:
1. Navigate to the target folder and activate the environment
2. Run the training script
3. Run the testing script

Testing Only:
1. Navigate to the target folder and activate the environment
2. Run the testing script

Outputs：
Training: Summaries and checkpoints are saved in the model_ckpt/ folder
Testing: Results are saved in the output_results/ folder

Notes：
1. Download the dataset and save it to your local machine or server before running
2. Update the folder paths and dataset locations in the commands accordingly
